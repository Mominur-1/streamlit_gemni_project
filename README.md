# 📸 Streamlit Gemini Project

An AI-powered application that generates notes, audio notes, and quizzes from image screenshots using Google Gemini AI.

## ✨ Features

- 📝 **Generate Notes** - AI-powered note generation from images
- 🔊 **Audio Notes** - Convert notes to speech
- ❓ **Quiz Generator** - Create quizzes from images
- ✅ **Quiz Answers** - Get correct answers

## 🚀 Live Demo

**App URL:** [https://mominur-ai-notes.streamlit.app](https://mominur-ai-notes.streamlit.app)

Click the link above to try the app!

## 🛠️ Technologies Used

- Streamlit
- Google Gemini AI
- gTTS (Google Text to Speech)
- Python

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/Mominur-1/streamlit_gemni_project.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set your API key:
```bash
export GEMINI_API_KEY=your_api_key_here
```

4. Run the app:
```bash
streamlit run app.py
```

## 📝 How to Use

1. Upload an image screenshot
2. Click "Generate Notes" to create AI notes
3. Click "Convert to Audio" to hear the notes
4. Select difficulty level and click "Generate Quiz"
